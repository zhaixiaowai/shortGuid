# shortGuid

使用JS实现无损缩短guid/uuid的长度到22位字符

using JS to achieve lossless reduction of the length of the GUID/UUID to 22 characters.

 

测试项目创建
npm install

测试项目运行
node ./main.js

测试项目访问
http://127.0.0.1:3456/
